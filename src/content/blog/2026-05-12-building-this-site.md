Every author needs a home on the web — somewhere that isn't rented ground on a social platform that could change the rules tomorrow. So I built one.

## The Stack

This site runs on [Astro](https://astro.build), a framework built for content-first websites. It ships zero JavaScript by default, renders everything to static HTML, and lets me write posts in plain Markdown files that I own forever. Styling is handled by Tailwind CSS. The whole thing is hosted for free on GitHub Pages.

## Why Build It Myself?

I could have used WordPress, Squarespace, or a dozen other platforms. But I wanted full ownership. The content lives in Markdown files in a Git repository. If Astro disappears tomorrow, the words survive. If GitHub Pages goes away, I can deploy the same static files anywhere.

For an author, the writing is the thing. Everything else should serve it.

## What's Here

Right now, the site has:

- A blog for development logs and writing updates
- A stories section for short fiction
- A books page (empty for now — that changes soon)
- The Steadfast Universe landing page
- RSS feed and sitemap for discoverability
- Dark and light mode, because everyone deserves a choice

## What's Next

Real content. The placeholder pages need to become real pages. The Steadfast Universe needs its proper introduction. And most importantly — the stories themselves need to start appearing here.

Stay tuned.
